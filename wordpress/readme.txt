Lets you create a quiz using your data.

== Description ==
Quizzer is a WordPress plugin that shows a series of questions and
possible answers. The results are shown on a separate page.

== Installation ==
1. Unzip the "quizzer.zip" file.
2. Upload the "quizzer" folder to the "/wp-content/plugins/" directory.
3. Activate the plugin through the "Plugins" menu in WordPress.
4. The questions will appear on any page with the shortcode [quiz-questions] in the content.
5. The results will appear on any page with the shortcode [quiz-results] in the content.
